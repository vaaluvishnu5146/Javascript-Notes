Types of functions

  • Pure Functions
  • Anonymous Functions
  • IIFE Functions
  • FLAT Arrow Functions

Pure Function

greet();
function greet() { return 'Hey'; }
  • It will get hoisted
  • Uses function keyword

IIFE Function

(IMMIDIATELY INVOKING FUNCTION EXPRESSION)
(() => { console.log('Hey') })();
  • It will not get hoisted
  • It calls itself

Anonymous Function

() => { console.log('Hey') }
  • A function without a name and defination
  • It will not get hoisted
  • If we need to send a function as a callbackm, then we can use Anonymous function

FLAT Arrow Function

const greet = () => { console.log('Hey') }
  • It will not get hoisted
  • If an anonymous function gets a name then it becomes FLAT Arrow function
