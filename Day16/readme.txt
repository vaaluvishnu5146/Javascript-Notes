What is Synchronous or blocking technique:

As javascript is an interpretter language it will execute code line-by-line, which makes other or upcoming lines of code to wait.
example: Waiter serving 1 table at a time.

What is ASYCHNRONOUS or NON-BLOCKING?

A technique that helps javascript to execute program in the non-blocking way or execute any piece of code parallely.
